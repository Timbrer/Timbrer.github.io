python学习手册(上)

Posted on 2018-06-06

Python语言

       Python是一种强类型动态语言，强类型语言运行时不会出现未定义行为，不允许错误（缓冲区溢出等或者类型不同）。而弱类型语言会允许一些错误的发生而保持程序继续运行（容忍隐式的类型转换等）。动态类型与静态类型是指这些错误在编译时被指出还是在运行时被指出。

动态类型

      Python的变量是不带类型的，而类型属于对象，将一个对象赋值给一个变量的时候实际上建立了一种引用的关系，以后操作这个变量就代表操作赋值给它的对象
      如果把一个变量赋值给一个新的变量，那么这两个变量名将指向同一个对象，相当于共享引用。而将一个变量通过操作符或函数产生的对象赋值给一个新的变量，变量将指向这个新创造的对象，而不是指向同一个对象(isnot)，即使它们相等(==)。这样子可以实现拷贝([:]、copy())
      当一个变量指向新的对象时，原来的对象将被回收

迭代协议

      迭代协议是指有next方法的对象会前进到下一结果，到末尾时引发StopIteration。
      iter()可以为可迭代对象生成一个迭代器，通过next()方法迭代
      可迭代对象都可以在for中使用，os.popen()的结果也可以迭代
      列表解析表达式可以嵌套循环[x+y for x in str1 for y in str2]表示y在内层循环
     许多内置函数都支持可迭代协议，比如sum、any、all、min、max和生成可迭代对象的map、filter、enumerate、zip等
      map、filter、enumerate、zip是自己的迭代器，在迭代一次之后就用尽了，list()方法也会用尽这些迭代器，它们不可以保存多个位置不同的迭代器，而range()不是这样的。

数据类型

      数据类型的转化直接使用type(obj)

数字

      +-*/**代表基本运算，数的精度自动扩充，不等号可以连用，如果第一个是false后面的不会看
      abs(),min(),max(),pow()提供了一些操作，math模块提供更丰富的操作math.pi,math.sin等，numpy和scipy库提供了科学计算相关方法
      random模块产生随机数random.random(),random.choice(list)等，此外还包括复数(a+bj)，固定精度数(小数)，有理数，分数，布尔值，矩阵，向量等对象
      三元操作符是x if y else z，or、and、not是逻辑或和非，in代表属于，/返回是真除法，//是截断除法，向下取整，math.trunc(a/b)才是和C++一样的整数除法
      进制转换oct(),hex(),bin(),ord()，还可以使用int(str,进制)实现任意进制的转化

集合

      集合是没有键的字典，同样不可放入可变对象
      使用set()函数和可迭代对象来创建，花括号和列表解析表达式也可以
      集合间可以使用+，-，位运算(or,and)，不等号(子集)，in来进行各种运算
      使用add()添加元素，update()相当于先set()再取并集，remove()移除元素

序列（包含字符串，列表，元组）

      使用len()验证长短，使用[num]访问元素，num为负自动加上len(list)，但是不可越界
      使用[n1:n2]表示分片处理[n1:n2)的元素，缺省时表示[0,-1]
      使用max(),min()求极值(str不可以)使用加法和乘法合并或重复元素

字符串’’

      不可变性，每个字符串操作都只能被定义为新的字符串作为其结果
      +可以拼接字符串，*可以让字符串重复多次
      使用find(str)方法查找特定字符串，返回首字符index
      使用replace(from,to)方法替换特定字符串
      使用str.split(char)方法将字符串以char分割为列表，与’str’.join(iter)方法互逆
      使用strip()方法去掉空白字符
      使用str%tuple或者“{num}”.format(str,)实现格式化替代
      使用map(ord,S)将整体转ascii码
      反斜线机制可以转义，在字符串前加r可以取消转义机制
      \0表示一个空字符，不再表示字符串结尾，使用ord()可以查看ascii码
      使用re.match(restr,str)实现正则的模式匹配
      使用encode(‘UTF-8’)方法进行解码，decode()与之相对

列表[]

      使用append()在末尾添加元素，remove()按照值删除元素
      使用insert(i,x)在任意位置添加元素，pop(i)删除任意位置元素
      使用sort()从小到大排序，reverse()翻转可得到从大到小的结果
      使用extend(list)让列表相加，使用count(x)统计某元素出现次数
      使用dellist[i]也可以删除元素，使用index(x,start=0,end=-1)得到元素的坐标
      使用[row[1] for row in M if row[1]%2==0]的列表解析式可以得到矩阵row中每行第一个
      求元素中为偶数的元素的列表，元祖，集合，字典都可以用这种方法，内置函数map()可以起到相同的作用
      L*n是将列表内的元素在原来列表内复制，[L]*n是生成多维数组的方法
      列表可以循环定义append()，即自身包含自身的引用

字典{key:value}

      使用dict(zip(iter1,iter2))可以快速创建字典，zip()是将迭代器打包为元祖列表的函数
      关键字形式dict(a=0,b=0)也可以创建字典对新的字典key赋值会创建这个key-value对
      使用keys(),values(),copy(),pop()下标或key,del D[key]可以实现相应功能
      使用update(D)可以合并字典

元祖（）

      不支持任何方法的调用，相当于不可修改的列表，是对数据类型的一种补充，相当于const类型的变量
      使用逗号可以创建元祖，元祖可以作为字典的键而列表不可以

文件（）

      使用open(file,method)生成文件对象，str可以加r不转义，str默认是‘r’(读)，还有‘rb’(二进制文件)，还可以加encoding参数解码
      使用read()方法读取到字符串中，可以加参数N读取前N个字符
使用readline()方法读取一行，readlines()方法生成字符串列表
      使用wirte()方法写入字符串，使用writelines()方法写入字符串列表，使用seek(N)方法移动文件中”指针”的位置，使用flush()方法可以将缓冲区内容写入

python语法

赋值/IO

      赋值语句建立对象的引用
      变量名在首次赋值时被创建，变量名引用前必须先赋值
      模块的导入，函数和类的定义，for循环变量以及函数参数都是隐式赋值运算
      使用分解赋值可以同时给多个变量赋值，a,b=’a’,’b’或[a,b]=[‘a’,’b’]，还有扩展的序列解包a,*b=’spam’，带*号永远是赋值一个列表，使用range()可以赋值连续整数
      前后有下划线的变量名是系统定义的，对解释器有特殊意义
      输入使用input()逐行读入，使用sys.stdin可以当作文件对象一样来处理，或者使用文件流
      输入使用print()函数，可选的参数有end=””(末尾默认加换行符)，file实现重定向，sep实现多个元素间的间隔
      使用StringIO将字符串转化为可以操作的文件对象，通过write()方法可以向流中添加内容，指针在尾部，使用seek(pos)可以跳转

分支/循环

      使用\可以续行
      if不使用括号，数字零、空对象、以及特殊None被认为是，使用elif代替elseif
      while后面可以接else，在循环结束且没遇到break的时候生效
      除了continue和break，新增加了pass做空白占位符，可以保持函数为空而不产生错误
      for循环可以遍历可迭代对象，对字典而言遍历key
      可以通过zip()函数同时遍历多个序列，到最短的那个序列之后的元祖都为空
      使用enumerate()可以形成(下标,元素)组成的列表

python函数

      函数也是一个对象，它允许添加任意的属性到记录信息里，也可以赋值给别的变量名，而c++的函数仅仅是一个代码段
      def语句是一个可执行的语句，在它运行的时候，它创建一个新的函数对象并将其赋值给一个变量名。因此def语句可以嵌套在别的语句中，比如if或者def自身等

作用域

      全局的是指在一个模块的顶层，外部模块的全局变量只是一个模块对象的属性
      当引用一个变量时，先查找本地变量，再查找上一级函数的本地变量，再查找全局变量，最后查找内置变量
      在def内部，任何对变量的赋值都认为是创建一个新的本地变量，如果想修改
      外部的变量，需要使用global(全局)语句或者nonlocal(嵌套函数作用域)语句声明变量，嵌套函数作用域在嵌套函数返回后仍然有效
      最好不要定义过多的全局变量，并尽可能减少文件间变量的修改，用函数参数和返回值代替，这样可以减少函数间和文件间的依赖

参数

      参数的传递是通过自动将对象赋值给本地变量实现的，不可变参数(整数、字符串等)通过“值”进行传递，可变对象(列表、字典)通过“指针”进行传递
      函数的参数匹配有多种方式，可以通过位置，或者关键字func(name=value)，也可以输入一个列表func(*seq)或者字典(**dict)，将被自动解析为单个的参数。函数声明时也有多种方法，可以使用def func(name=value)设置默认值，可以func(*name)\func(**name)自动收集所有参数，并保存为元祖\字典

高级用法

      函数作为一个对象，有许多内置的属性，同样的也可以添加任意的用户定义的属性。在python3中函数可以添加注解，def func(a:’str’,..)→int:
      lambda表达式(lambdaarg1=value,arg2,…argN:expression)。lambda的主题是一个单一的表达式，而不是代码块。可以把lambda赋值当成函数来用，它只能是实现有限的逻辑，不允许嵌套。
      map函数map(func,list)。用来对可迭代对象分别进行操作，然后集合在一起，可以通过tuple或者list获取结果。filter与其类似。
      reduce函数readuce(func,list)。对list中的元素迭代的调用func函数，前一个函数的返回结果会成为下一次调用的参数。

MIPS快速指南

Posted on 2017-10-11

      主要总结自MPIS Quick Turotial，本文内容会随着学习的深入补充。

数据类型与表示

      - MIPS指令都为32位
      - 一个字节(byte)8位，半字(halfword)16位，字(word)32位
      - 字符类型需要一个字节，整数类型需要四个字节
      - 字符用单引号括起，字符串用双引号括起

寄存器

      - 共有32个通用寄存器
      - 指令里使用寄存器要在前面加$
      - 使用寄存器可以用编号也可以用名字 e.g. $0, $t1
      - 专用寄存器lo和hi不能直接寻址，只有特殊的指令可以访问
      - 栈是从高地址到低地址
 
      通用寄存器具体内容和命名这里不再赘述，最基本的寄存器有$v0寄存器用来存储输入输出等指令，$a0-$a3寄存器多用做$v0所存操作的参数，$s0-$s7寄存器用来存值(类似全局变量)，$t0-$t9用来存临时值(类似局部变量)。

程序结构

      - 程序是以.asm或.s结尾的文件
      - 数据段以.data独占一行开始，第二行开始声明变量
      - 数据段中声明的变量都存储在内存里
      - 代码段以.text独占一行开始
      - 代码段中可以有多个标签(类似c中的函数头)，程序从标签mian:开始执行
      - 程序在main中执行exit的syscall后结束
      - #是注释符号，类似c中的//，没有多行引用

e.g. hello world

.text
main:
    la $a0 str   # 将str的地址加载到$a0寄存器中
    li $v0 4     # 将$v0寄存器加载4号指令，4号指令的参数为$a0中的值
    syscall       # 执行$v0中的指令

    li $v0 10     # 将$v0寄存器加载10号指令
    syscall       # 执行$v0中的指令
.data
    str: .asciiz "hello world\n"

      下面我们开始讲解基本的指令

变量声明

      - 变量声明的格式为: name: storage_type value
      - 声明的存储类型有.word .byte .space，分别是字，字节和连续空间(类似数组)

e.g.

var1:       .word   3         # 一个值为3的整数变量
array1:     .byte   'a','b'   # 一个含有两个字符的数组
array2:     .space  40        # 一个40字节长的空白区域，可以当作长度为40的字符数组也可以当作长度为10的整数数组

加载存储指令

load：

      - lw(lb)指令 lw(lb) register_destination, RAM_source
      - 将内存中的数据加载到寄存器中，lw(load word)拷贝四个字节，lb(load byte)拷贝一个字节

store：

      - sw(sb)指令 sw(sb) register_source, RAM_destination
      - 将寄存器中的数据存储到内存中，sw(store word)存储四个字节，lb(store byte)存储一个字节

load immediate：

      - li指令 li register_destination, value
      - 将立即数value存储到寄存器中

e.g.

.data
    var1: .word 1
    char1: .byte 'a'
.text
    lw $t0, var1   # 将整数var1加载到$t0寄存器
    lb $t1, char1  # 将字符char1加载到$t1寄存器
    li $t2, 5      # 将5存储到寄存器$t2中
    sw $t2, var1   # 将$t2寄存器中的值赋给var1

间接寻址与基址寻址

      - la命令 la register_destination, RAM_address
      - 将数据(变量)的地址加载到寄存器中

      下面通过例子来说明如何间接寻址和基址寻址

e.g.

.data
    array1:  .space  12   # 声明12字节长的空间用来存储三个整数
.text
    la $t0, array1        # 将array的首地址加载到寄存器$t0中
    li $t1, 5             # 将立即数5加载到寄存器$t1中
    sw $t1, ($t0)         # 将寄存器$t1中的值存储到寄存器$t0存储的地址所代表的内存位置里(即array1[0] = $t1）
    li $t1, 13            # 将立即数13加载到寄存器$t1中
    sw $t1, 4($t0)        # array[1] = $t1, 一个整数占四个字节，所以向后偏移四个字节
    li $t1, -7            # $t1 = -7
    sw $t1, 8($t0)        # array[2] = $t1,同上

算术指令

      算术命令可以在MIPS Instruction Refence中找到更具体的介绍，这里先介绍几个基本的指令

      - 算术指令大多有三个操作数
      - 所有的操作数都必须是寄存器，不能简介寻址或者来自内存的变量

add    $t0,$t1,$t2    #  $t0 = $t1 + $t2  加法指令

sub    $t2,$t3,$t4    #  $t2 = $t3 - $t4  减法指令

addi    $t2,$t3, 5    #  $t2 = $t3 + 5;   立即数加法 ps.不存在立即数减法

addu    $t1,$t6,$t7    #  $t1 = $t6 + $t7;   无符号数加法

subu    $t1,$t6,$t7    #  $t1 = $t6 - $t7;   无符号数减法

mult    $t3,$t4       # 将两个32位数相乘得到一个64位数，加载在lo和Hi寄存器中

                      # 高位在Hi寄存器中，加载在Lo寄存器中

div    $t5,$t6      #  $t5 / $t6, 将商加载在Hi寄存器中，将余数加载在Lo寄存器中

mfhi    $t0        #  $t0 = Hi 将专用寄存器Hi中的值加载到通用寄存器$t0中

mflo    $t1        #  $t1 = Lo 将专用寄存器Lo中的值加载到通用寄存器$t1中

sll     $t0.$t1.value        #  $t0 = $t1 << value  左移value的位运算

srl     $t2.$t3.value        #  $t2 = $t3 >> value  右移value的位运算

控制结构

分支命令

beq    $t0,$t1,target    #  如果 $t0 = $t1 跳转到一个标签
blt    $t0,$t1,target    #  如果 $t0 < $t1 跳转到一个标签
ble    $t0,$t1,target    #  如果 $t0 <= $t1 跳转到一个标签
bgt    $t0,$t1,target    #  如果 $t0 > $t1 跳转到一个标签
bge    $t0,$t1,target    #  如果 $t0 >= $t1 跳转到一个标签
bne    $t0,$t1,target    #  如果 $t0 <> $t1 跳转到一个标签

e.g. dead-loop hello world

.text
main:
    la $a0 str   # 将str的地址加载到$a0寄存器中
    li $v0 4     # 将$v0寄存器加载4号指令，4号指令的参数为$a0中的值
    syscall       # 执行$v0中的指令
    beq $v0 $v0 main   #跳回main标签重新进行，程序陷入死循环，不断输出hello world.
.data
    str: .asciiz "hello world\n"
跳转命令
j     target  # 无条件直接跳转到一个标签
jr    $t3     # 跳转到$t3存储的地址
字程序调用(类似函数)

      - jal指令 jal label
      - 跳转到label，并且将下一条指令的地址加载到$ra寄存器中

      - jr指令 jr $ra
      - 通常如此使用，跳到$ra寄存器储存的地址的指令

       这里值得注意的是，此处的方法不可以在字程序(类似函数)中再调用新的字程序

系统调用

      - 用来执行输入输出操作和退出程序等操作
      - 使用syscall执行系统调用
      - 使用前要先在寄存器$v0,$a0,$a1中加载合适的值
      - 系统调用的返回值加载在$v0寄存器中

服务 $v0中的数字 参数 结果
输出整数 1 $a0加载要输出的整数
输出浮点数 2 $f12加载要输出的浮点数
输出双精度数 3 $f12加载要输出的浮点数
输出字符串 4 $a0加载要输出的串的首地址
读取整数 5 整数返回$v0
读取浮点数 6 浮点数返回$v0
读取双精度数 7 双精度数返回$v0
读取字符串 8 $a0加载读取字符串的首地址
$a1加载字符串长度
申请内存空间 9 $a0加载要申请的内存大小 返回的地址加载到$v0
退出 10
