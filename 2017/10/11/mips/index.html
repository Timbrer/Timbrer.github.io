MIPS快速指南

Posted on 2017-10-11

      主要总结自MPIS Quick Turotial，本文内容会随着学习的深入补充。

数据类型与表示

      - MIPS指令都为32位
      - 一个字节(byte)8位，半字(halfword)16位，字(word)32位
      - 字符类型需要一个字节，整数类型需要四个字节
      - 字符用单引号括起，字符串用双引号括起

寄存器

      - 共有32个通用寄存器
      - 指令里使用寄存器要在前面加$
      - 使用寄存器可以用编号也可以用名字 e.g. $0, $t1
      - 专用寄存器lo和hi不能直接寻址，只有特殊的指令可以访问
      - 栈是从高地址到低地址
 
      通用寄存器具体内容和命名这里不再赘述，最基本的寄存器有$v0寄存器用来存储输入输出等指令，$a0-$a3寄存器多用做$v0所存操作的参数，$s0-$s7寄存器用来存值(类似全局变量)，$t0-$t9用来存临时值(类似局部变量)。

程序结构

      - 程序是以.asm或.s结尾的文件
      - 数据段以.data独占一行开始，第二行开始声明变量
      - 数据段中声明的变量都存储在内存里
      - 代码段以.text独占一行开始
      - 代码段中可以有多个标签(类似c中的函数头)，程序从标签mian:开始执行
      - 程序在main中执行exit的syscall后结束
      - #是注释符号，类似c中的//，没有多行引用

e.g. hello world

.text
main:
    la $a0 str   # 将str的地址加载到$a0寄存器中
    li $v0 4     # 将$v0寄存器加载4号指令，4号指令的参数为$a0中的值
    syscall       # 执行$v0中的指令

    li $v0 10     # 将$v0寄存器加载10号指令
    syscall       # 执行$v0中的指令
.data
    str: .asciiz "hello world\n"

      下面我们开始讲解基本的指令

变量声明

      - 变量声明的格式为: name: storage_type value
      - 声明的存储类型有.word .byte .space，分别是字，字节和连续空间(类似数组)

e.g.

var1:       .word   3         # 一个值为3的整数变量
array1:     .byte   'a','b'   # 一个含有两个字符的数组
array2:     .space  40        # 一个40字节长的空白区域，可以当作长度为40的字符数组也可以当作长度为10的整数数组

加载存储指令

load：

      - lw(lb)指令 lw(lb) register_destination, RAM_source
      - 将内存中的数据加载到寄存器中，lw(load word)拷贝四个字节，lb(load byte)拷贝一个字节

store：

      - sw(sb)指令 sw(sb) register_source, RAM_destination
      - 将寄存器中的数据存储到内存中，sw(store word)存储四个字节，lb(store byte)存储一个字节

load immediate：

      - li指令 li register_destination, value
      - 将立即数value存储到寄存器中

e.g.

.data
    var1: .word 1
    char1: .byte 'a'
.text
    lw $t0, var1   # 将整数var1加载到$t0寄存器
    lb $t1, char1  # 将字符char1加载到$t1寄存器
    li $t2, 5      # 将5存储到寄存器$t2中
    sw $t2, var1   # 将$t2寄存器中的值赋给var1

间接寻址与基址寻址

      - la命令 la register_destination, RAM_address
      - 将数据(变量)的地址加载到寄存器中

      下面通过例子来说明如何间接寻址和基址寻址

e.g.

.data
    array1:  .space  12   # 声明12字节长的空间用来存储三个整数
.text
    la $t0, array1        # 将array的首地址加载到寄存器$t0中
    li $t1, 5             # 将立即数5加载到寄存器$t1中
    sw $t1, ($t0)         # 将寄存器$t1中的值存储到寄存器$t0存储的地址所代表的内存位置里(即array1[0] = $t1）
    li $t1, 13            # 将立即数13加载到寄存器$t1中
    sw $t1, 4($t0)        # array[1] = $t1, 一个整数占四个字节，所以向后偏移四个字节
    li $t1, -7            # $t1 = -7
    sw $t1, 8($t0)        # array[2] = $t1,同上

算术指令

      算术命令可以在MIPS Instruction Refence中找到更具体的介绍，这里先介绍几个基本的指令

      - 算术指令大多有三个操作数
      - 所有的操作数都必须是寄存器，不能简介寻址或者来自内存的变量

add    $t0,$t1,$t2    #  $t0 = $t1 + $t2  加法指令

sub    $t2,$t3,$t4    #  $t2 = $t3 - $t4  减法指令

addi    $t2,$t3, 5    #  $t2 = $t3 + 5;   立即数加法 ps.不存在立即数减法

addu    $t1,$t6,$t7    #  $t1 = $t6 + $t7;   无符号数加法

subu    $t1,$t6,$t7    #  $t1 = $t6 - $t7;   无符号数减法

mult    $t3,$t4       # 将两个32位数相乘得到一个64位数，加载在lo和Hi寄存器中

                      # 高位在Hi寄存器中，加载在Lo寄存器中

div    $t5,$t6      #  $t5 / $t6, 将商加载在Hi寄存器中，将余数加载在Lo寄存器中

mfhi    $t0        #  $t0 = Hi 将专用寄存器Hi中的值加载到通用寄存器$t0中

mflo    $t1        #  $t1 = Lo 将专用寄存器Lo中的值加载到通用寄存器$t1中

sll     $t0.$t1.value        #  $t0 = $t1 << value  左移value的位运算

srl     $t2.$t3.value        #  $t2 = $t3 >> value  右移value的位运算

控制结构

分支命令

beq    $t0,$t1,target    #  如果 $t0 = $t1 跳转到一个标签
blt    $t0,$t1,target    #  如果 $t0 < $t1 跳转到一个标签
ble    $t0,$t1,target    #  如果 $t0 <= $t1 跳转到一个标签
bgt    $t0,$t1,target    #  如果 $t0 > $t1 跳转到一个标签
bge    $t0,$t1,target    #  如果 $t0 >= $t1 跳转到一个标签
bne    $t0,$t1,target    #  如果 $t0 <> $t1 跳转到一个标签

e.g. dead-loop hello world

.text
main:
    la $a0 str   # 将str的地址加载到$a0寄存器中
    li $v0 4     # 将$v0寄存器加载4号指令，4号指令的参数为$a0中的值
    syscall       # 执行$v0中的指令
    beq $v0 $v0 main   #跳回main标签重新进行，程序陷入死循环，不断输出hello world.
.data
    str: .asciiz "hello world\n"
跳转命令
j     target  # 无条件直接跳转到一个标签
jr    $t3     # 跳转到$t3存储的地址
字程序调用(类似函数)

      - jal指令 jal label
      - 跳转到label，并且将下一条指令的地址加载到$ra寄存器中

      - jr指令 jr $ra
      - 通常如此使用，跳到$ra寄存器储存的地址的指令

       这里值得注意的是，此处的方法不可以在字程序(类似函数)中再调用新的字程序

系统调用

      - 用来执行输入输出操作和退出程序等操作
      - 使用syscall执行系统调用
      - 使用前要先在寄存器$v0,$a0,$a1中加载合适的值
      - 系统调用的返回值加载在$v0寄存器中

服务 $v0中的数字 参数 结果
输出整数 1 $a0加载要输出的整数
输出浮点数 2 $f12加载要输出的浮点数
输出双精度数 3 $f12加载要输出的浮点数
输出字符串 4 $a0加载要输出的串的首地址
读取整数 5 整数返回$v0
读取浮点数 6 浮点数返回$v0
读取双精度数 7 双精度数返回$v0
读取字符串 8 $a0加载读取字符串的首地址
$a1加载字符串长度
申请内存空间 9 $a0加载要申请的内存大小 返回的地址加载到$v0
退出 10
